📊 Online Courses Analysis Dashboard – Power BI
🚀 Project Overview

This project presents an end-to-end Business Intelligence solution built using Power BI to analyze online courses data and generate actionable insights.

The objective was to transform raw educational data into a structured, interactive, and decision-support dashboard that helps stakeholders understand market demand, pricing strategies, subscriber behavior, and growth trends.

The dashboard follows a clean, user-friendly design and applies best practices in data modeling, DAX calculations, and data storytelling.

🎯 Business Objectives

The main goals of this analysis were to:

Identify the most in-demand course subjects

Analyze the relationship between pricing and subscriber engagement

Compare free versus paid course performance

Track course growth trends over time

Highlight top-performing courses based on subscribers and reviews

Provide data-driven insights for better business decisions

This project simulates a real-world scenario for education platforms and digital learning providers.

🧭 Dashboard Structure

The report is divided into three interactive pages:

🔹 1. Overview Page

This page provides a high-level summary of overall performance and key metrics:

Total Courses

Total Lectures

Total Subscribers

Total Reviews

Subscriber Distribution by Subject

Yearly Course Growth Trend

Top Courses by Subscribers

📈 Key Insight:
Web Development dominates subscriber count, indicating strong and consistent market demand compared to other subjects.

🔹 2. Price Analysis Page

This page focuses on pricing strategies and user behavior patterns:

Average Price by Subject

Price Categories Distribution

Price vs Content Duration Analysis

Free vs Paid Courses Comparison

Engagement & Review Analysis

📊 Key Insights:

Free courses attract a larger number of users.

Paid courses show stronger engagement and higher-quality reviews.

Pricing impacts perceived value and subscriber behavior.

🔹 3. Details Page

This page enables detailed course-level analysis with interactive filters and drill-down capabilities:

Course-level performance metrics

Subject & Level filtering

Dynamic slicing

Comparative analysis

This page allows decision-makers to identify improvement opportunities and top-performing content areas.
